What is Hacktoberfest?

Hacktoberfest is for everyone. Whether it’s your first time—or your ninth—it’s almost time to hack out four pristine pull/merge requests and complete your mission for open source. Join other members of the open-source community on the Hacktoberfest Discord.

OPEN SOURCE/Hacktoberfest Best Practices

-> Select a project You are intrested in
-> read the README.md
-> setup that project locally
-> read CONTRIBUTING.md
-> go to issues and select one
-> read its convo and study
-> try to recreate the issue and now work on it
-> don't know something, search it online read come back try again repeat
-> don't get what a particular thing does just print it
-> find references for the particular function or element you are working on
